📱 Cómo editar microciclos antiguos ▾
¿Qué es el localStorage?Todos los datos de la app se guardan en el navegador en un objeto JSON con clave b26v2. Para editarlo, usas la consola del navegador — no hace falta instalar nada.
Paso 1 — Abrir la consola · En Chrome/Safari móvil: conecta el móvil al PC, abre chrome://inspect y selecciona la pestaña.
· En PC: pulsa F12 → pestaña "Console".
Paso 2 — Ver todos los datos actuales
const d = JSON.parse(localStorage.getItem('b26v2')); console.log(JSON.stringify(d, null, 2));
Paso 3a — Editar peso medio de un micro completado
Por ejemplo, corregir el peso del M2 (índice 1 en wHist):
const d = JSON.parse(localStorage.getItem('b26v2')); d.wHist[1] = 74.5; // cambia el peso del M2 localStorage.setItem('b26v2', JSON.stringify(d)); location.reload();
Paso 3b — Editar calorías medias de un micro
const d = JSON.parse(localStorage.getItem('b26v2')); d.cHist[0] = 3100; // M1 = índice 0 localStorage.setItem('b26v2', JSON.stringify(d)); location.reload();
Paso 3c — Editar kilos/reps de un ejercicio en un snapshot
Los snapshots guardan las sesiones completas. Por ejemplo, editar el press banca del M1 (snapshot índice 0), primera serie:
const d = JSON.parse(localStorage.getItem('b26v2')); // snapshots[0] = M1, sessions[0] = Empuje 1, exercises[0] = Press banca d.snapshots[0].sessions[0].exercises[0].sets[0].kg = 87.5; d.snapshots[0].sessions[0].exercises[0].sets[0].reps = 6; localStorage.setItem('b26v2', JSON.stringify(d)); location.reload();
Paso 3d — Editar el microciclo ACTUAL (en curso)
const d = JSON.parse(localStorage.getItem('b26v2')); // sessions[2] = Pierna 1, exercises[1] = Sentadilla d.sessions[2].exercises[1].sets[0].kg = 45; localStorage.setItem('b26v2', JSON.stringify(d)); location.reload();
Paso 3e — Corregir una entrada del log de peso/calorías
const d = JSON.parse(localStorage.getItem('b26v2')); d.log['D3'] = {w: 73.8, c: 3150}; // sobrescribe día 3 localStorage.setItem('b26v2', JSON.stringify(d)); location.reload();
🔢 Índices de sesiones (sessions[])
0 = Empuje 1 · 1 = Tracción 1 · 2 = Pierna 1 · 3 = Empuje 2 · 4 = Tracción 2 · 5 = Pierna 2
💡 Tip: ver el índice de un ejercicio
const d = JSON.parse(localStorage.getItem('b26v2')); d.sessions[0].exercises.forEach((e,i) => console.log(i, e.name));
🛡 Antes de editar, haz una copia de seguridad
copy(localStorage.getItem('b26v2')); // Pega en un .txt para tener backup
